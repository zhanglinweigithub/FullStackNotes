# Feign

# дёҖгҖҒFeignиҝңзЁӢи°ғз”Ё

е…ҲжқҘзңӢжҲ‘д»¬д»ҘеүҚеҲ©з”ЁRestTemplateеҸ‘иө·иҝңзЁӢи°ғз”Ёзҡ„д»Јз Ғпјҡ

еӯҳеңЁдёӢйқўзҡ„й—®йўҳпјҡ

  • д»Јз ҒеҸҜиҜ»жҖ§е·®пјҢзј–зЁӢдҪ“йӘҢдёҚз»ҹдёҖ

  • еҸӮж•°еӨҚжқӮURLйҡҫд»Ҙз»ҙжҠӨ

FeignжҳҜдёҖдёӘеЈ°жҳҺејҸзҡ„httpе®ўжҲ·з«ҜпјҢе®ҳж–№ең°еқҖпјҡhttps://github.com/OpenFeign/feign

е…¶дҪңз”Ёе°ұжҳҜеё®еҠ©жҲ‘д»¬дјҳйӣ…зҡ„е®һзҺ°httpиҜ·жұӮзҡ„еҸ‘йҖҒпјҢи§ЈеҶідёҠйқўжҸҗеҲ°зҡ„й—®йўҳгҖӮ

# FeignдҪҝз”Ё

Feginзҡ„дҪҝз”ЁжӯҘйӘӨеҰӮдёӢпјҡ

# 1пјүеј•е…Ҙдҫқиө–

жҲ‘д»¬еңЁorder-serviceжңҚеҠЎзҡ„pomж–Үд»¶дёӯеј•е…Ҙfeignзҡ„дҫқиө–пјҡ

<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
</dependency>

# 2пјүж·»еҠ жіЁи§Ј

еңЁorder-serviceзҡ„еҗҜеҠЁзұ»ж·»еҠ жіЁи§ЈејҖеҗҜFeignзҡ„еҠҹиғҪпјҡ

# 3пјүзј–еҶҷFeignзҡ„е®ўжҲ·з«Ҝ

еңЁorder-serviceдёӯж–°е»әдёҖдёӘжҺҘеҸЈпјҢеҶ…е®№еҰӮдёӢпјҡ

// ж·»еҠ  @FeignClient жіЁи§ЈпјҢжҢҮе®ҡиҰҒи°ғз”Ёе“ӘдёӘжңҚеҠЎзҡ„жҺҘеҸЈ
@FeignClient("userservice")
public interface UserClient {
    @GetMapping("/user/{id}")
    User findById(@PathVariable("id") Long id);
}

жіЁж„Ҹпјҡи·Ҝеҫ„иҰҒеҶҷиў«и°ғз”ЁжҺҘеҸЈзҡ„е…Ёи·Ҝеҫ„пјҲзұ»и·Ҝеҫ„ + ж–№жі•и·Ҝеҫ„пјү

иҝҷдёӘе®ўжҲ·з«Ҝдё»иҰҒжҳҜеҹәдәҺSpringMVCзҡ„жіЁи§ЈжқҘеЈ°жҳҺиҝңзЁӢи°ғз”Ёзҡ„дҝЎжҒҜпјҢжҜ”еҰӮпјҡ

  • жңҚеҠЎеҗҚз§°пјҡuserservice
  • иҜ·жұӮж–№ејҸпјҡGET
  • иҜ·жұӮи·Ҝеҫ„пјҡ/user/{id}
  • иҜ·жұӮеҸӮж•°пјҡLong id
  • иҝ”еӣһеҖјзұ»еһӢпјҡUser

иҝҷж ·пјҢFeignе°ұеҸҜд»Ҙеё®еҠ©жҲ‘д»¬еҸ‘йҖҒhttpиҜ·жұӮпјҢж— йңҖиҮӘе·ұдҪҝз”ЁRestTemplateжқҘеҸ‘йҖҒдәҶгҖӮ

# 4пјүжөӢиҜ•

жҳҜдёҚжҳҜзңӢиө·жқҘдјҳйӣ…еӨҡдәҶгҖӮ

# 5пјүжҖ»з»“

дҪҝз”ЁFeignзҡ„жӯҘйӘӨпјҡ

в‘  еј•е…Ҙдҫқиө–

в‘Ў еҗҜеҠЁзұ»ж·»еҠ @EnableFeignClientsжіЁи§Ј

в‘ў зј–еҶҷFeignClientжҺҘеҸЈпјҢж·»еҠ  @FeignClient жіЁи§ЈпјҢжҢҮе®ҡиҰҒи°ғз”Ёе“ӘдёӘжңҚеҠЎзҡ„жҺҘеҸЈ

в‘Ј дҪҝз”ЁFeignClientдёӯе®ҡд№үзҡ„ж–№жі•д»ЈжӣҝRestTemplate

# иҮӘе®ҡд№үй…ҚзҪ®

FeignеҸҜд»Ҙж”ҜжҢҒеҫҲеӨҡзҡ„иҮӘе®ҡд№үй…ҚзҪ®пјҢеҰӮдёӢиЎЁжүҖзӨәпјҡ

зұ»еһӢдҪңз”ЁиҜҙжҳҺ
feign.Logger.Levelдҝ®ж”№ж—Ҙеҝ—зә§еҲ«еҢ…еҗ«еӣӣз§ҚдёҚеҗҢзҡ„зә§еҲ«пјҡNONEгҖҒBASICгҖҒHEADERSгҖҒFULL
feign.codec.Decoderе“Қеә”з»“жһңзҡ„и§ЈжһҗеҷЁhttpиҝңзЁӢи°ғз”Ёзҡ„з»“жһңеҒҡи§ЈжһҗпјҢдҫӢеҰӮи§Јжһҗjsonеӯ—з¬ҰдёІдёәjavaеҜ№иұЎ
feign.codec.EncoderиҜ·жұӮеҸӮж•°зј–з Ғе°ҶиҜ·жұӮеҸӮж•°зј–з ҒпјҢдҫҝдәҺйҖҡиҝҮhttpиҜ·жұӮеҸ‘йҖҒ
feign. Contractж”ҜжҢҒзҡ„жіЁи§Јж јејҸй»ҳи®ӨжҳҜSpringMVCзҡ„жіЁи§Ј
feign. RetryerеӨұиҙҘйҮҚиҜ•жңәеҲ¶иҜ·жұӮеӨұиҙҘзҡ„йҮҚиҜ•жңәеҲ¶пјҢй»ҳи®ӨжҳҜжІЎжңүпјҢдёҚиҝҮдјҡдҪҝз”ЁRibbonзҡ„йҮҚиҜ•

дёҖиҲ¬жғ…еҶөдёӢпјҢй»ҳи®ӨеҖје°ұиғҪж»Ўи¶іжҲ‘д»¬дҪҝз”ЁпјҢеҰӮжһңиҰҒиҮӘе®ҡд№үж—¶пјҢеҸӘйңҖиҰҒеҲӣе»әиҮӘе®ҡд№үзҡ„@BeanиҰҶзӣ–й»ҳи®ӨBeanеҚіеҸҜгҖӮ

дёӢйқўд»Ҙж—Ҙеҝ—дёәдҫӢжқҘжј”зӨәеҰӮдҪ•иҮӘе®ҡд№үй…ҚзҪ®гҖӮ

# й…ҚзҪ®ж–Үд»¶ж–№ејҸ

еҹәдәҺй…ҚзҪ®ж–Үд»¶дҝ®ж”№feignзҡ„ж—Ҙеҝ—зә§еҲ«еҸҜд»Ҙй’ҲеҜ№еҚ•дёӘжңҚеҠЎпјҡ

feign:  
  client:
    config: 
      userservice: # й’ҲеҜ№жҹҗдёӘеҫ®жңҚеҠЎзҡ„й…ҚзҪ®
        loggerLevel: FULL #  ж—Ҙеҝ—зә§еҲ«

д№ҹеҸҜд»Ҙй’ҲеҜ№жүҖжңүжңҚеҠЎпјҡ

feign:  
  client:
    config: 
      default: # иҝҷйҮҢз”Ёdefaultе°ұжҳҜе…ЁеұҖй…ҚзҪ®пјҢеҰӮжһңжҳҜеҶҷжңҚеҠЎеҗҚз§°пјҢеҲҷжҳҜй’ҲеҜ№жҹҗдёӘеҫ®жңҚеҠЎзҡ„й…ҚзҪ®
        loggerLevel: FULL #  ж—Ҙеҝ—зә§еҲ«

иҖҢж—Ҙеҝ—зҡ„зә§еҲ«еҲҶдёәеӣӣз§Қпјҡ

  • NONEпјҡдёҚи®°еҪ•д»»дҪ•ж—Ҙеҝ—дҝЎжҒҜпјҢиҝҷжҳҜй»ҳи®ӨеҖјгҖӮ
  • BASICпјҡд»…и®°еҪ•иҜ·жұӮзҡ„ж–№жі•пјҢURLд»ҘеҸҠе“Қеә”зҠ¶жҖҒз Ғе’Ңжү§иЎҢж—¶й—ҙ
  • HEADERSпјҡеңЁBASICзҡ„еҹәзЎҖдёҠпјҢйўқеӨ–и®°еҪ•дәҶиҜ·жұӮе’Ңе“Қеә”зҡ„еӨҙдҝЎжҒҜ
  • FULLпјҡи®°еҪ•жүҖжңүиҜ·жұӮе’Ңе“Қеә”зҡ„жҳҺз»ҶпјҢеҢ…жӢ¬еӨҙдҝЎжҒҜгҖҒиҜ·жұӮдҪ“гҖҒе…ғж•°жҚ®гҖӮ

# Javaд»Јз Ғж–№ејҸ

д№ҹеҸҜд»ҘеҹәдәҺJavaд»Јз ҒжқҘдҝ®ж”№ж—Ҙеҝ—зә§еҲ«пјҢе…ҲеЈ°жҳҺдёҖдёӘзұ»пјҢз„¶еҗҺеЈ°жҳҺдёҖдёӘLogger.Levelзҡ„еҜ№иұЎпјҡ

public class DefaultFeignConfiguration  {
    @Bean
    public Logger.Level feignLogLevel(){
        return Logger.Level.BASIC; // ж—Ҙеҝ—зә§еҲ«дёәBASIC
    }
}

еҰӮжһңиҰҒе…ЁеұҖз”ҹж•ҲпјҢе°Ҷе…¶ж”ҫеҲ°еҗҜеҠЁзұ»зҡ„@EnableFeignClientsиҝҷдёӘжіЁи§Јдёӯпјҡ

@EnableFeignClients(defaultConfiguration = DefaultFeignConfiguration .class)

еҰӮжһңжҳҜеұҖйғЁз”ҹж•ҲпјҢеҲҷжҠҠе®ғж”ҫеҲ°еҜ№еә”зҡ„@FeignClientиҝҷдёӘжіЁи§Јдёӯпјҡ

@FeignClient(value = "userservice", configuration = DefaultFeignConfiguration .class)

# дәҢгҖҒFeignдҪҝз”ЁдјҳеҢ–

Feignеә•еұӮеҸ‘иө·httpиҜ·жұӮпјҢдҫқиө–дәҺе…¶е®ғзҡ„жЎҶжһ¶гҖӮе…¶еә•еұӮе®ўжҲ·з«Ҝе®һзҺ°еҢ…жӢ¬пјҡ

вҖўURLConnectionпјҡй»ҳи®Өе®һзҺ°пјҢдёҚж”ҜжҢҒиҝһжҺҘжұ 

вҖўApache HttpClient пјҡж”ҜжҢҒиҝһжҺҘжұ 

вҖўOKHttpпјҡж”ҜжҢҒиҝһжҺҘжұ 

еӣ жӯӨжҸҗй«ҳFeignзҡ„жҖ§иғҪдё»иҰҒжүӢж®өе°ұжҳҜдҪҝз”ЁиҝһжҺҘжұ д»Јжӣҝй»ҳи®Өзҡ„URLConnectionгҖӮ

иҝҷйҮҢжҲ‘д»¬з”ЁApacheзҡ„HttpClientжқҘжј”зӨәгҖӮ

# 1пјүеј•е…Ҙдҫқиө–

еңЁorder-serviceзҡ„pomж–Үд»¶дёӯеј•е…ҘApacheзҡ„HttpClientдҫқиө–пјҡ

<!--httpClientзҡ„дҫқиө– -->
<dependency>
    <groupId>io.github.openfeign</groupId>
    <artifactId>feign-httpclient</artifactId>
</dependency>

# 2пјүй…ҚзҪ®иҝһжҺҘжұ 

еңЁorder-serviceзҡ„application.ymlдёӯж·»еҠ й…ҚзҪ®пјҡ

feign:
  client:
    config:
      default: # defaultе…ЁеұҖзҡ„й…ҚзҪ®
        loggerLevel: BASIC # ж—Ҙеҝ—зә§еҲ«пјҢBASICе°ұжҳҜеҹәжң¬зҡ„иҜ·жұӮе’Ңе“Қеә”дҝЎжҒҜ
  httpclient:
    enabled: true # ејҖеҗҜfeignеҜ№HttpClientзҡ„ж”ҜжҢҒ
    max-connections: 200 # жңҖеӨ§зҡ„иҝһжҺҘж•°
    max-connections-per-route: 50 # жҜҸдёӘи·Ҝеҫ„зҡ„жңҖеӨ§иҝһжҺҘж•°

жҺҘдёӢжқҘпјҢеңЁFeignClientFactoryBeanдёӯзҡ„loadBalanceж–№жі•дёӯжү“ж–ӯзӮ№пјҡ

Debugж–№ејҸеҗҜеҠЁorder-serviceжңҚеҠЎпјҢеҸҜд»ҘзңӢеҲ°иҝҷйҮҢзҡ„clientпјҢеә•еұӮе°ұжҳҜApache HttpClientпјҡ

# жҖ»з»“

Feignзҡ„дјҳеҢ–пјҡ

1.ж—Ҙеҝ—зә§еҲ«е°ҪйҮҸз”Ёbasic

2.дҪҝз”ЁHttpClientжҲ–OKHttpд»ЈжӣҝURLConnection

в‘  еј•е…Ҙfeign-httpClientдҫқиө–

в‘Ў й…ҚзҪ®ж–Үд»¶ејҖеҗҜhttpClientеҠҹиғҪпјҢи®ҫзҪ®иҝһжҺҘжұ еҸӮж•°

# дёүгҖҒжңҖдҪіе®һи·ө

жүҖи°“жңҖиҝ‘е®һи·өпјҢе°ұжҳҜдҪҝз”ЁиҝҮзЁӢдёӯжҖ»з»“зҡ„з»ҸйӘҢпјҢжңҖеҘҪзҡ„дёҖз§ҚдҪҝз”Ёж–№ејҸгҖӮ

иҮӘд№ и§ӮеҜҹеҸҜд»ҘеҸ‘зҺ°пјҢFeignзҡ„е®ўжҲ·з«ҜдёҺжңҚеҠЎжҸҗдҫӣиҖ…зҡ„controllerд»Јз Ғйқһеёёзӣёдјјпјҡ

feignе®ўжҲ·з«Ҝпјҡ

UserControllerпјҡ

жңүжІЎжңүдёҖз§ҚеҠһжі•з®ҖеҢ–иҝҷз§ҚйҮҚеӨҚзҡ„д»Јз Ғзј–еҶҷе‘ўпјҹ

# з»§жүҝж–№ејҸ

дёҖж ·зҡ„д»Јз ҒеҸҜд»ҘйҖҡиҝҮз»§жүҝжқҘе…ұдә«пјҡ

1пјүе®ҡд№үдёҖдёӘAPIжҺҘеҸЈпјҢеҲ©з”Ёе®ҡд№үж–№жі•пјҢе№¶еҹәдәҺSpringMVCжіЁи§ЈеҒҡеЈ°жҳҺгҖӮ

2пјүFeignе®ўжҲ·з«Ҝе’ҢControllerйғҪйӣҶжҲҗиҜҘжҺҘеҸЈ

дјҳзӮ№пјҡ

  • з®ҖеҚ•
  • е®һзҺ°дәҶд»Јз Ғе…ұдә«

зјәзӮ№пјҡ

  • жңҚеҠЎжҸҗдҫӣж–№гҖҒжңҚеҠЎж¶Ҳиҙ№ж–№зҙ§иҖҰеҗҲ

  • еҸӮж•°еҲ—иЎЁдёӯзҡ„жіЁи§Јжҳ е°„е№¶дёҚдјҡз»§жүҝпјҢеӣ жӯӨControllerдёӯеҝ…йЎ»еҶҚж¬ЎеЈ°жҳҺж–№жі•гҖҒеҸӮж•°еҲ—иЎЁгҖҒжіЁи§Ј

# жҠҪеҸ–ж–№ејҸ

е°ҶFeignзҡ„ClientжҠҪеҸ–дёәзӢ¬з«ӢжЁЎеқ—пјҢе№¶дё”жҠҠжҺҘеҸЈжңүе…ізҡ„POJOгҖҒй»ҳи®Өзҡ„Feignй…ҚзҪ®йғҪж”ҫеҲ°иҝҷдёӘжЁЎеқ—дёӯпјҢжҸҗдҫӣз»ҷжүҖжңүж¶Ҳиҙ№иҖ…дҪҝз”ЁгҖӮ

дҫӢеҰӮпјҢе°ҶUserClientгҖҒUserгҖҒFeignзҡ„й»ҳи®Өй…ҚзҪ®йғҪжҠҪеҸ–еҲ°дёҖдёӘfeign-apiеҢ…дёӯпјҢжүҖжңүеҫ®жңҚеҠЎеј•з”ЁиҜҘдҫқиө–еҢ…пјҢеҚіеҸҜзӣҙжҺҘдҪҝз”ЁгҖӮ

# е®һзҺ°еҹәдәҺжҠҪеҸ–зҡ„жңҖдҪіе®һи·ө

# 1пјүжҠҪеҸ–

йҰ–е…ҲеҲӣе»әдёҖдёӘmoduleпјҢе‘ҪеҗҚдёәfeign-apiпјҡ

йЎ№зӣ®з»“жһ„пјҡ

еңЁfeign-apiдёӯз„¶еҗҺеј•е…Ҙfeignзҡ„starterдҫқиө–

<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-openfeign</artifactId>
</dependency>

з„¶еҗҺпјҢorder-serviceдёӯзј–еҶҷзҡ„UserClientгҖҒUserгҖҒDefaultFeignConfigurationйғҪеӨҚеҲ¶еҲ°feign-apiйЎ№зӣ®дёӯ

# 2пјүеңЁorder-serviceдёӯдҪҝз”Ёfeign-api

йҰ–е…ҲпјҢеҲ йҷӨorder-serviceдёӯзҡ„UserClientгҖҒUserгҖҒDefaultFeignConfigurationзӯүзұ»жҲ–жҺҘеҸЈгҖӮ

еңЁorder-serviceзҡ„pomж–Үд»¶дёӯдёӯеј•е…Ҙfeign-apiзҡ„дҫқиө–пјҡ

<dependency>
    <groupId>cn.itcast.demo</groupId>
    <artifactId>feign-api</artifactId>
    <version>1.0</version>
</dependency>

дҝ®ж”№order-serviceдёӯзҡ„жүҖжңүдёҺдёҠиҝ°дёүдёӘз»„д»¶жңүе…ізҡ„еҜјеҢ…йғЁеҲҶпјҢж”№жҲҗеҜје…Ҙfeign-apiдёӯзҡ„еҢ…

# 3пјүйҮҚеҗҜжөӢиҜ•

йҮҚеҗҜеҗҺпјҢеҸ‘зҺ°жңҚеҠЎжҠҘй”ҷдәҶпјҡ

иҝҷжҳҜеӣ дёәUserClientзҺ°еңЁеңЁcn.itcast.feign.clientsеҢ…дёӢпјҢ

иҖҢorder-serviceзҡ„@EnableFeignClientsжіЁи§ЈжҳҜеңЁcn.itcast.orderеҢ…дёӢпјҢдёҚеңЁеҗҢдёҖдёӘеҢ…пјҢж— жі•жү«жҸҸеҲ°UserClientгҖӮ

# 4пјүи§ЈеҶіжү«жҸҸеҢ…й—®йўҳ

ж–№ејҸдёҖпјҡ

жҢҮе®ҡFeignеә”иҜҘжү«жҸҸзҡ„еҢ…пјҡ

@EnableFeignClients(basePackages = "cn.itcast.feign.clients")

ж–№ејҸдәҢпјҡ

жҢҮе®ҡйңҖиҰҒеҠ иҪҪзҡ„ClientжҺҘеҸЈпјҡ

@EnableFeignClients(clients = {UserClient.class})
